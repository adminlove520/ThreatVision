关于 ThreatVision

系统简介

ThreatVision 是一个自动化安全监控与AI分析系统，旨在实时追踪最新的安全漏洞（CVE）和 GitHub 上的安全相关仓库，并利用人工智能技术进行深度分析。

系统 7x24 小时不间断运行，自动完成安全信息收集、分析和报告生成，并通过多种渠道发布安全情报。

核心功能

  • 多源数据监控（CVE、GitHub仓库）
  • 智能AI分析（OpenAI & Gemini双引擎）
  • 自动化工作流（7x24小时运行）
  • 每日安全简报生成与发布
  • Redis消息队列优化API调用
